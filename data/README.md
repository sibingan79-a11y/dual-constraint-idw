# Data

`example_rainfall_stations.csv` is a processed and anonymized rainfall-station dataset for reproducing the algorithmic workflow.

Columns:

- `station_id`: anonymized station identifier;
- `x_km`: relative x coordinate in kilometers;
- `y_km`: relative y coordinate in kilometers;
- `rain_mm_h`: hourly rainfall intensity in mm h-1.

Station names and original station identifiers are not included in this public example dataset.
